# Pixi-fps

Easy to use fps counter for pixi.js

Works with pixi v4 or higher
![alt text](./img/example.png)

## Installation

```javascript
npm install pixi-fps
```

## Usage

```javascript
import PixiFps from "pixi-fps";

const fpsCounter = new PixiFps();

stage.addChild(fpsCounter);
```
